HOW IT WORKS
═════════════════════════════════════════════════════════════════════════════

1. USER ASKS QUESTION
   "I need practice with recursion"

2. SYSTEM RECOMMENDS
   [Using AI + Rules + Feedback adjustments]
   [Shows 5 recommended problems]

3. USER WORKS ON PROBLEM
   [Solves it, marks as completed]

4. SYSTEM COLLECTS FEEDBACK
   "Was that helpful? [Y/n]"
   "How difficult? [easy/medium/hard]"
   "Time spent? [minutes]"

5. SYSTEM LEARNS
   [Updates skill profile]
   [Adjusts future recommendations]

6. NEXT RECOMMENDATIONS ARE SMARTER ✨
   [Feedback immediately influences results]

KEY FEATURES
═════════════════════════════════════════════════════════════════════════════

✅ Feedback Collection
   → Automatic prompts for helpful/unhelpful ratings
   → Difficulty assessment per problem
   → Time tracking for performance analysis

✅ Skill Profiling
   → Topic strengths/weaknesses (0.0-1.0 scale)
   → Difficulty comfort levels (easy/medium/hard)
   → Recommendation quality tracking
   → Learning velocity detection

✅ Adaptive Recommendations
   → Weak topics automatically boosted
   → Strong topics automatically reduced
   → Difficulty adjusted based on comfort level
   → All adjustments applied instantly

✅ Learning Insights
   → Strengths visualization with progress bars
   → Improvement areas with priority ranking
   → Difficulty progression recommendations
   → Learning velocity indicators

✅ Local Privacy
   → All data stored in data/ folder
   → Nothing sent to cloud (except optional AI API)
   → User fully controls all data
   → Can reset anytime by deleting data/

═════════════════════════════════════════════════════════════════════════════

NEW COMMANDS
═════════════════════════════════════════════════════════════════════════════

insights   → View learning dashboard & skill profile
stats      → View progress statistics  
rescan     → Rescan problem files
exit       → Quit application

Feedback is collected automatically:
  • After marking attempted: "Rate this recommendation?"
  • After marking completed: "Time spent? Helpful?"

QUICK START
═════════════════════════════════════════════════════════════════════════════

   cd problem_recommender
   pip install -r requirements.txt

2. Run
   python3 main.py

   Run "insights" after ~10 problems

4. Watch it improve
   Recommendations get better over time!
